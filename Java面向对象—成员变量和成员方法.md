# 成员变量和成员方法

======================================

* 变量声明的位置决定变量作用域

* 变量作用域确定可在程序中按变量名访问该变量的区域

* 成员变量和局部变量的区别：

  * 作用域不同：
    * 局部变量的作用域仅限于定义它的方法
    * 成员变量的作用域在整个类内部都是可见的
  * 初始值不同：
    * Java会给成员变量一个初始值
    * Java不会给局部变量赋予初始值
  * 在同一个方法中，不允许有同名局部变量
  * 在不同的方法中，可以有同名局部变量
  * 两类变量同名时，局部变量具有更高的优先级

* 方法调用

  | Student类的方法a()调用Teacher类的方法c()，先创建类对象，然后使用"."调用 |
